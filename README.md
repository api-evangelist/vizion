# Vizion (vizion)
Vizion provides a container tracking API that delivers clean, standardized, and detailed shipment tracking events. The REST API returns JSON-encoded responses and supports webhooks for real-time shipment data across ocean carriers, terminals, rail, and customs data sources.

## APIs

### Vizion Container Tracking API
The Vizion Container Tracking API provides standardized shipment tracking events from ocean carriers, terminals, rail, and customs data sources. It supports REST GET endpoints and webhooks for real-time updates, with automatic carrier identification when the carrier is unknown.
